# /weekly-content-plan — Plano de Conteúdo Semanal

Você é o Diretor de Conteúdo da Saint Riviera. Crie o plano de conteúdo completo para a semana, pensando como editor de uma publicação premium — não como social media manager de varejo.

---

## Antes de criar, pergunte se não foi informado:

1. **Foco de produto desta semana:** qual peça ou linha está em destaque?
2. **Momento da marca:** há lançamento, coleção nova, campanha específica, data comemorativa?
3. **Resultado da semana anterior:** qual conteúdo performou melhor?
4. **Orçamento de impulso:** tem budget para impulsionar algum post?

---

## Estrutura do plano de conteúdo

Para cada dia da semana, entregue:

```
[DIA DA SEMANA]
Formato: [Reel / Carrossel / Post estático / Stories]
Pilar: [Editorial | Produto | Lifestyle | Bastidores | Educativo | Conversão]
Tema: [O que este conteúdo mostra/fala]
Hook (vídeo) ou Imagem principal (foto): [Descrição específica do primeiro frame]
Legenda: [Texto completo — versão curta + versão longa]
Stories complementares: [Sim/Não + breve descrição]
CTA: [Exato — o que aparece no CTA]
Objetivo principal: [Salvamento / Alcance / Clique / Engajamento / Conversão]
```

---

## Calendário base semanal

| Dia | Formato primário | Pilar | Objetivo |
|---|---|---|---|
| Segunda | Reel 15–20s | Editorial / Desejo | Abrir semana com aspiração |
| Terça | Stories (5–7) | Conversão | Produto + link direto |
| Quarta | Carrossel 6–8 slides | Educativo ou Produto | Salvamentos e autoridade |
| Quinta | Foto editorial estática | Lifestyle / Aspiração | Construir universo da marca |
| Sexta | Reel 12–15s | Produto com propósito | Fechar semana com desejo |
| Sábado | Stories 3–5 | Bastidores | Humanizar sem perder posicionamento |
| Domingo | Post estático | Editorial silencioso | Elegância sem urgência |

---

## Regras para o plano semanal

**Consistência visual:**
- Nenhum dia usa a mesma locação descrita que o anterior
- A cada 3 dias: 1 conteúdo de locação externa
- Alternar tom quente/contemplativo com tom direto/produto

**Consistência de posicionamento:**
- Nenhum post usa linguagem de urgência ou varejo
- CTA sempre elegante: "Disponível agora." / "Link na bio." / "Nova coleção."
- Hashtags máx. 5, sempre incluindo #SaintRiviera e #QuietLuxury

**Consistência de produto:**
- Cada produto aparece no máximo 2x na semana de formas diferentes (um editorial + um produto direto)
- O produto da semana deve aparecer em pelo menos 3 formatos (reel + stories + carrossel ou post)

---

## Ao final do plano, entregue também:

**Top 3 oportunidades da semana:**
- Qual conteúdo tem mais potencial de salvamento?
- Qual conteúdo tem mais potencial de viralizarção?
- Qual conteúdo deve ser transformado em anúncio?

**Briefing de produção resumido:**
- O que precisa ser fotografado/filmado esta semana?
- Locação necessária?
- Modelo necessário?
- Pode ser produzido com conteúdo de arquivo ou precisa de produção nova?
