# Guia de Produção — Stories Saint Riviera
## Presença e Percepção | Storyboard Executável

**Como usar:** abra este arquivo, escolha a sequência do dia, siga campo por campo. Cada story tem instrução exata de o que gravar/fotografar, o que escrever e onde colocar.

---

## SEÇÃO 1 — Lógica das 3 Sequências

### 1. Para que serve cada sequência

**Sequência Masculina (7 stories)**
Para homens de 32–50 anos que já sentiram que foram julgados antes de abrir a boca. Essa sequência ativa essa memória, nomeia a dor, apresenta a roupa como solução e fecha com produto. É a sequência com maior probabilidade de gerar venda direta — o argumento ressoa com quem se importa com percepção profissional.

**Sequência Feminina (7 stories)**
Para mulheres de 28–45 anos que estão cansadas de "precisar se esforçar para ser percebida". Valida esse cansaço, propõe elegância contida como alternativa e fecha com o Conjunto Riviera Blanc. Converte melhor quem já segue a marca ou já visitou a loja.

**Sequência Mista (8 stories)**
Para qualquer pessoa — homem ou mulher — que ainda não conhece bem a marca. É a sequência de abertura porque a ideia "sua roupa fala antes de você" é universal. Constrói posicionamento antes de falar de produto.

---

### 2. Quando postar cada sequência

| Dia | Sequência | Motivo |
|---|---|---|
| **Hoje (quinta)** | Mista | Abre o universo para qualquer público |
| **Sexta** | Masculina | Público profissional, dia de trabalho |
| **Sábado** | Feminina | Fins de semana têm mais mulheres navegando |

**Nunca as três no mesmo dia.** Cada sequência precisa de espaço para ser consumida.

---

### 3. Respostas diretas

| Pergunta | Resposta |
|---|---|
| Qual vende mais hoje? | **Masculina** — story 3 é o gatilho mais forte |
| Qual constrói mais marca? | **Mista** — estabelece a ideia-mãe |
| Qual gera mais direct? | **Feminina** — stories de identidade geram mais resposta emocional |
| Devo postar todas de uma vez? | **Não** — separe por dia |
| Qual é mais fácil de produzir hoje? | **Mista** — é mais baseada em texto |

# SEQUÊNCIA MASCULINA — 7 Stories
**Tema:** "O homem é lido antes de ser ouvido."
**Produto principal:** Conjunto Classic Navy (Polo Tricô + Calça Alfaiataria)

---

### M1 — O Espelho

| Campo | Instrução |
|---|---|
| **Objetivo** | Fazer o cliente se reconhecer: "já senti isso" |
| **Tipo de mídia** | Vídeo 5s, câmera lenta (slow motion) |
| **O que aparece** | Mãos masculinas ajustando colarinho de camisa social branca. Luz de janela entrando pela esquerda. Fundo levemente desfocado (parede branca ou madeira clara) |
| **Texto exato** | "Antes de você falar, a sala já decidiu." |
| **Posição do texto** | Inferior esquerdo — 75% da altura, 8% da largura |
| **Tamanho** | Grande — ocupa ~50% da largura do frame |
| **Tipografia** | Playfair Display Italic |
| **Cor do texto** | Off-white #F5F0E8 |
| **Fundo/efeito** | Sem overlay. Se a área do vídeo onde o texto vai não for escura, adicione uma faixa preta de 20% opacidade APENAS atrás do texto |
| **Legenda do story** | Nenhuma |
| **Sticker** | Nenhum |
| **CTA** | Nenhum |
| **Produto mostrado** | Camisa Social Slim — Branco Cru (a mão que aparece usa ela) |
| **Intenção psicológica** | Ativar memória de situação onde aparência decidiu antes da fala |
| **O que sentir** | "Conheço esse sentimento. É real." |
| **Como ajuda a vender** | Não vende — cria tensão emocional que os próximos stories resolvem |

**Como produzir na prática:**
Coloque a camisa em alguém (ou em você mesmo na frente de um espelho). Posicione ao lado de uma janela com luz natural. Grave em slow motion (modo câmera lenta do iPhone ou Android). Enquadre do ombro para baixo — o rosto não precisa aparecer. Foque nas mãos ajustando o colarinho. Corte para 5 segundos.

# SEQUÊNCIA FEMININA — 7 Stories
**Tema:** "A mulher elegante não precisa disputar atenção. Ela é percebida."

| Campo | Instrução |

| **Sticker** | Nenhum |
| **CTA** | Nenhum |
| **Produto mostrado** | Camisa de Linho Lavado (as mãos usam ela) |

|---|---|
